# 听力葵花宝典 - 听写地狱 + 变速突破

> 针对听力"全链条问题"（识别→速度→记忆）的 4 周速成计划

---

## 诊断结果

```
问题 A: 听不懂单词 — 即使认识的词，听到也反应不过来
问题 B: 跟不上速度 — 单个词能懂，连成句子就丢了
问题 C: 无法记住 — 听的时候懂，说完就忘了

根本原因: 听力输入几乎为零，听力系统处于"休眠状态"
```

---

## 整体框架

### 4 周突破计划

```
Week 1-2: 听写地狱期
├── 目标：暴力定位所有"听不出"的词
├── 方法：每天 30 分钟高强度听写
└── 产出：建立「听力黑名单」

Week 3-4: 变速突破期
├── 目标：突破速度瓶颈 + 强化记忆
├── 方法：1.25x → 1.5x 变速 + shadowing
└── 产出：能跟上正常语速对话
```

### 每日 30 分钟结构

```
0-5 min    热身：复习昨天的「黑名单词」
5-20 min   核心：听写训练（地狱本体）
20-28 min  巩固：shadowing 跟读
28-30 min  记录：更新黑名单 + 标记进度
```

### 材料选择

- **主教材**：国家地理 Listening and Notetaking（Level 2 或 3）
- **补充**：21st Century Communication 的技术相关单元
- **工作场景**：后期加入真实会议/技术播客片段

## 听写地狱 - 具体操作

**核心原则：不是"听大意"，是"每个词都要抓出来"**

### Step 1：选材料（2 分钟音频）

- 从 Listening and Notetaking 选一段对话/独白
- 第一周选慢速材料，第二周选正常语速
- 每段材料用 2-3 天（Day1 生听，Day2 精磨，Day3 巩固）

### Step 2：听写流程（15 分钟）

```
第 1 遍：盲听
├── 不暂停，听完整段
├── 写下能听到的所有词（哪怕只是碎片）
└── 标记完全听不懂的位置（用 ??? 标记）

第 2 遍：断句听
├── 每句暂停，尽量补全
├── 听 3 遍还是听不出的词 → 标红
└── 不要超过 3 遍，避免"假装听懂"

第 3 遍：核对原文
├── 对照 transcript，找出所有错误
├── 分类标记：
│   🔴 完全没听出 → 加入黑名单
│   🟡 听错了 → 记录"我听成 X，实际是 Y"
│   🟢 听对了 → 跳过
```

### Step 3：黑名单记录格式

```markdown
| 日期 | 原词/短语 | 我听成 | 原因分析 | 状态 |
|------|-----------|--------|----------|------|
| 1/28 | would've  | wood   | 连读弱化 | 🔴   |
| 1/28 | architecture | archi-tech-cher | 重音错 | 🔴 |
```

**原因分析**常见类型：
- **连读**（would have → would've）
- **弱读**（to/the/a 几乎消失）
- **重音位置错**
- **单词本身不认识**
- **认识但声音和拼写没建立映射**

## Shadowing 跟读 + 变速训练

### Shadowing（影子跟读）- 每天 8 分钟

解决「听懂但记不住」(问题 C) 的核心武器。

**操作方法：**

```
Level 1（第 1 周）：看文本跟读
├── 播放音频，眼睛看着 transcript
├── 嘴巴同步跟读，延迟 0.5 秒
└── 目标：跟上节奏，模仿语调

Level 2（第 2 周）：盲跟
├── 不看文本，纯靠耳朵
├── 跟不上就跳过，继续跟下一句
└── 目标：强迫大脑"实时处理"

Level 3（第 3-4 周）：变速盲跟
├── 1.25x 速度跟读
├── 适应后升到 1.5x
└── 目标：当正常语速变得"很慢"
```

### 变速训练 - 邪道核心

**原理**：当你能跟上 1.5x 速度后，正常语速会感觉像慢动作。

```
工具：任何播放器的变速功能
├── Mac: IINA / VLC
├── 手机: Podcast app / YouTube
└── 在线: YouTube 自带变速

训练节奏：
Week 3: 1.0x → 1.25x（适应期）
Week 4: 1.25x → 1.5x（突破期）

注意：
- 听不懂 1.25x？退回 1.0x 再精听
- 能轻松跟 1.5x？尝试 1.75x（地狱模式）
```

### 每日 Shadowing 流程（8 分钟）

```
0-3 min: 用今天听写的材料，看文本跟读 2 遍
3-6 min: 盲跟 2 遍（允许跟丢）
6-8 min: 加速到 1.25x 跟 1 遍（Week 3 开始）
```

## 材料分层 + 进阶路径

### 材料分层（由易到难）

```
Layer 1（Week 1-2）：教材音频
├── 国家地理 Listening and Notetaking Level 2
├── 特点：语速慢、发音清晰、有 transcript
└── 目的：建立基础听力回路

Layer 2（Week 3-4）：半真实材料
├── 21st Century Communication 技术单元
├── TED-Ed 短视频（5 分钟以内，有字幕）
└── 目的：过渡到真实语速

Layer 3（Week 5+）：真实工作场景
├── 技术 Podcast（推荐见下）
├── YouTube 技术频道
├── 真实会议录音（如果有）
└── 目的：直接对接工作目标
```

### 推荐的技术 Podcast / YouTube（适合前端）

**入门级（语速较慢，表达清晰）：**
- **Syntax.fm** — 两个前端开发者聊天，口语化但清晰
- **JS Party** — 适中语速，话题贴近工作

**进阶级（正常语速，多人讨论）：**
- **The Changelog** — 技术访谈，多种口音
- **React Podcast** — React 生态相关

**YouTube 频道：**
- **Fireship** — 短视频，语速快但有字幕
- **Theo - t3.gg** — 真实前端讨论，语速快

### 4 周后的进阶路径

```
Week 5-8：维持 + 扩展
├── 每天 20 分钟听 Syntax.fm（通勤/碎片时间也可）
├── 每周选 1 集做精听（周末 30 分钟）
├── 持续更新黑名单
└── 目标：能听懂 70% 以上技术讨论

Week 9-12：实战检验
├── 尝试听真实会议录音
├── 参加英语技术直播/Q&A
└── 目标：能实时跟上同事讨论
```

## 第一周启动清单

### 今天就做（10 分钟）

```
□ 找到教材：国家地理 Listening and Notetaking Level 2
  └── 确保有音频 + transcript

□ 准备工具
  ├── 播放器：IINA 或 VLC（支持变速、快捷键暂停）
  ├── 记录：新建文件夹 ~/english-practice/listening/
  └── 模板：创建 daily-log.md 和 blacklist.md

□ 选定练习时间
  └── 每天固定时段 ________（建议早上或 iTalki 课前）
```

### Day 1 具体任务

```
材料：Listening and Notetaking Unit 1 第一段对话
时长：选 1-2 分钟片段

执行：
0-5 min   播放 1 遍，写下听到的碎片
5-15 min  断句听写，每句最多 3 遍
15-20 min 核对 transcript，标记所有错误
20-28 min 看文本跟读 3 遍 + 盲跟 2 遍
28-30 min 记录黑名单（至少 5 个词）
```

### 第一周节奏

```
Day 1-2：Unit 1 片段 A（同一段材料用 2 天磨透）
Day 3-4：Unit 1 片段 B
Day 5-6：Unit 2 片段 A
Day 7：  复习整周黑名单 + 测试（重听 Day 1 材料，看进步）
```

### 成功标准

```
Week 1 结束时，你应该：
✓ 建立了稳定的每日 30 分钟习惯
✓ 黑名单上有 30-50 个词
✓ 同一段材料盲听正确率从 ~40% 提升到 ~80%
✓ 能跟上 1.0x 语速的 shadowing
```

## 常见问题

**Q: 听写太痛苦，想放弃怎么办？**
A: 这就是"地狱"的意义。痛苦 = 在学习。降低材料难度，但不要降低标准。

**Q: 黑名单越来越长怎么办？**
A: 正常。前两周会爆炸式增长，第三周开始会有词升级到 🟡🟢。

**Q: 没有 transcript 的材料怎么办？**
A: 用 Whisper 生成，或者只选有字幕的材料。

**Q: 每天 30 分钟不够怎么办？**
A: 够了。听力是"高强度短时间"比"低强度长时间"更有效。
