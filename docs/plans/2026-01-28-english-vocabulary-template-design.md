# English Vocabulary Learning Template Design

> 设计日期: 2026-01-28
> 目的: 为英语学习创建统一的词汇/短语/语法概念学习模板

---

## 设计背景

### 输入材料
1. `todo/01_Goroutine与Channel.md` - 技术文档，结构清晰
2. `todo/01_ostensible.md` - 词汇学习文档，实战丰富
3. `todo/template.md` - 原有模板

### 设计目标
糅合两份文档的优点，创建一份适用于以下内容的统一模板：
- Vocabulary words (单词)
- Phrasal verbs (短语动词)
- Collocations (搭配)
- Grammar concepts (语法概念)

---

## 模板结构 (10 Sections)

### Section 1: 【30字核心】用一句话说清本质
- 简洁定义 + 核心语义特征
- 包含词性标注
- 约30字

### Section 2: 【第一性原理】
- **2.1 词源追溯 (Etymology)**: 词根拆解、来源语言、原始含义
- **2.2 认知定位 (Cognitive)**: 这个词解决什么表达需求
- **2.3 语义演变**: 原始 → 中间 → 现代含义
- **2.4 一句话总结**

### Section 3: 【核心概念】(动态数量)
- 数量根据词性/含义动态调整
- 每个词性/含义使用 **CELTA 方法论**:
  - Meaning (语义): 定义、语境、例句
  - Form (形式): 词性、变形、句型
  - Pronunciation (发音): 音标、重音、常见发音错误
  - Appropriacy (得体性): 仅在相关时出现
- 包含: **Phrasal Verbs 短语动词表**
- 包含: **Collocations 常用搭配速查表**

### Section 4: 【最小可用】
- 必记含义 (1-2个最常用)
- 必记搭配 (3-5个)
- 必记句型
- 快速自检清单

### Section 5: 【类比】
- 使用前端开发领域类比
- 包含代码示例
- 类比对照表

### Section 6: 【反直觉点】
- 常见误区 (2-3个)
- 每个误区包含:
  - 为什么错
  - 为什么容易错
  - 正误对比例句

### Section 7: 【实战】
- **7.1 工作场景口语实践**: Standup, Code Review, PR, Meetings
- **7.2 正式语境示例**: 学术、新闻、商务邮件
- **7.3 即用型口语句型模板**
- **7.4 Do's and Don'ts 实战贴士**

### Section 8: 【面试必问】
- 普通回答 vs 出彩回答
- 结构化回答技巧
- 为什么出彩的分析

### Section 9: 【化骨绵掌】
- 完全灵活的知识卡片
- 数量和主题根据词/概念重点自由组织
- 常见主题参考: 词源、发音、词性、同义词、反义词、搭配、语境、错误、文化、应用等

### Section 10: 【公式总结】
- **使用公式**: 搭配结构
- **记忆公式**: 核心概念组合
- **对比公式**: 近义词区分
- **场景公式**: 适用/不适用场景
- **句型公式**: 常用句型模板

## 设计决策记录

| 决策点 | 选择 | 理由 |
|-------|------|------|
| 核心概念数量 | 动态 | 根据词性/含义灵活调整 |
| CELTA 详细程度 | Streamlined | M+F+P 一起，Appropriacy 按需 |
| 第一性原理内容 | Etymology + Cognitive | 双重理解更深入 |
| 实战范围 | Comprehensive | 工作 + 正式 + 口语模板 |
| 化骨绵掌结构 | 完全灵活 | 按词重点自由组织 |
| 公式总结类型 | 多种公式 | 使用/记忆/对比/场景/句型 |
| 文档语言 | 中文结构 + 英文例句 | 符合学习习惯 |
